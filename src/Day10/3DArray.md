### 3D Array ###
A 3D array is an extension of a 2D array, where each element is itself a 2D array. 
It can be visualized as a cube or a collection of matrices. 
A 3D array is defined by three dimensions: the number of layers (or depth), the number of rows, and the number of columns. 
This structure allows for the storage of more complex data, such as a series of images or a collection of tables.
